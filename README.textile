h1. NAME

Avocado

h1. DESCRIPTION

Avocado is syntactic sugar built on top of Plack that helps power the ohess.org website. Don't use this though, it was only a programming exercise. If you are masochistic, you can find the source code -- an example of Avocado usage -- on github.

You were warned.
